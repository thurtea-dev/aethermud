# staff-handbook/

In-depth staff reference for AetherMUD, split into 16 chapters plus an
index. This is player/staff-facing game-mechanics documentation (how the
Rifts systems work in this MUD), not repo setup/ops documentation - see
`docs/` for that.

- `index.md` - table of contents / entry point.
- `ch01-server.md` - server basics for staff.
- `ch02-gotchas.md` - known gotchas staff run into.
- `ch03-staff-ranks.md` - staff rank structure (mortal to apprentice to
  full role).
- `ch04-wizard-tools.md` - wizard tool usage.
- `ch05-chargen-flow.md` - character generation flow.
- `ch06-races.md` / `ch07-occs.md` - race and OCC (class) reference.
- `ch08-mdc-sdc.md` - Mega-Damage vs. Structural Damage systems.
- `ch09-combat.md` - combat mechanics.
- `ch10-skills.md` / `ch11-spells.md` / `ch12-psionics.md` - the three
  major ability systems.
- `ch13-ppe-isp.md` - magic/psionic energy pools.
- `ch14-alignments.md` / `ch15-insanities.md` - alignment and insanity
  systems.
- `ch16-builder.md` - building/content-creation reference for staff.
