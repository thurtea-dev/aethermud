# LLM breadcrumb - staff-handbook/

1. Read `/CLAUDE.md` at the repo root first.
2. Read `README.md` in this directory (above), or better, `index.md` for
   the actual table of contents.
3. Check the repo root's `LLM_BREADCRUMB.md` for workspace-wide
   conventions before assuming this directory's gotchas are
   self-contained.
4. Related: `www/*.html` contains an offline HTML version of some staff
   guides (QCS, coding, domain, roleplay, admin, chargen) - check both if
   researching a staff-facing topic, they may not be perfectly in sync.

## Gotchas specific to this directory

- This describes **game mechanics as implemented**, not aspirational
  design. If a chapter's description of a system disagrees with the
  actual LPC code under `nightmare3_fluffos_v2/lib/`, the code is
  authoritative - flag the doc as stale rather than trusting it blindly.
- Chapter numbering is meaningful (ch01-ch16) - if adding new staff
  documentation, prefer extending an existing chapter or appending a new
  numbered chapter over creating an unrelated top-level file.
- This is distinct from `docs/DEVELOPMENT.md` (contributor/LPC coding
  rules) and from CLAUDE.md (mandatory coding rules) - this handbook is
  about playing/staffing the game, not about editing its code.
