# internal/

Personal and machine-specific working notes. Not authoritative project
documentation - see `docs/` for that. Treat these as scratch/planning
material that may be stale, superseded, or written for one specific
machine.

- `tomorrow.md` - a running scratch file, often holding the next
  session's starting prompt/instructions. Frequently overwritten.
- `install-aethermud.md` - step-by-step notes for moving AetherMUD from
  WSL to a Fedora machine (machine-specific, not a general install guide
  - see `docs/INSTALL.md` for that).
- `TRANSFER-CHECKLIST.md` - a WSL-to-Fedora-Workstation transfer
  checklist generated from a specific audit.
- `report.md` - an older progress/remaining-work report; superseded by
  `docs/ASSESSMENT.md` for current state.
- `Playtest-list.md` - superseded; merged into
  `docs/playtest-checklist.md`. Kept for history, safe to delete once
  confirmed unused.
