Apprentice Wizard

Position: apprentice. Role tag: apprentice. Who tag: [Apprentice].

Apprentices are temporary wizards created by makewiz. They get a workroom, basic ambassador-level file tools, and an apprentice kit. The kit is the path to a permanent staff role.

The kit

Type kit (or apprentice) to open the tool:

1. Choose track (rp / domain / coding)
2. View task brief
3. Draft / edit submission
4. Submit to domain queue
5. View my submission status

Track tasks

  • RP: campaign/quest proposal, then domain review, then coding queue
  • Domain: area outline, then domain approve, then ready for setrole
  • Coding: realm room and NPC paths, then domain review, then coding check

What you can do

  • workroom, kit
  • ed / cat / ls (ambassador path)
  • cre channel
  • Read the reference book in your workroom

What you cannot do yet

  • QCS build (needs coding role and the Staff of Creation)
  • Domain dominion tools
  • RP skill / tattoo tools
  • Admin commands (update, warmboot, setrole, etc.)

Review and promotion

Domain wizards review submissions under domain option 10; coding wizards review the coding queue with review. Ready for setrole is a flag only; an admin must still run:

setrole <player> admin|domain|coding|rp
