About the game

Measured from the live mudlib (see ASSESSMENT.md, 2026-07-13):

  • 61 playable races, 62 OCCs
  • ~116 spells, ~50 psionic powers, 158 skills
  • ~430 playable Rifts rooms across Praxis, Chi-Town, Tolkeen, Horton, Splynn, New Camelot, and more
  • 173 mortal commands; custom prompt with MDC/SDC/PPE/ISP pool tokens (prompt, prompt tokens)

In-game help is two-level: help lists categories; help <category> lists topics; help <topic> opens a topic (separator-insensitive: armor of ithan matches armor_of_ithan). Use help index for the flat list.

Install on Fedora Workstation

The transfer package production-aethermud/ ships with the full archive and a Fedora-specific guide:

  • aethermud-full-YYYYMMDD.tar.gz and its .sha256
  • install-aethermud.md (HDD copy, verify, dnf deps, rebuild driver, start)
  • INSTALL.md, RUNNING.md, README.md

Short path: verify the checksum, extract into your home directory, rebuild the FluffOS driver on Fedora, then ./mud.sh start. Full steps are in install-aethermud.md.
