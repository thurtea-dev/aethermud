AetherMUD

AetherMUD, a TTRPG-inspired MUD on FluffOS / Nightmare III. Palladium Rifts on a living mudlib.

The Great Cataclysm shattered Earth's golden age, and the ley lines erupted with the magic of a dying world. Rifts opened to a thousand alien dimensions, and everything came through. Play a Coalition grunt, a Ley Line Walker, a Juicer burning bright, or a dragon hatchling wearing a borrowed human shape. Explore Chi-Town, Tolkeen, the Splynn Dimensional Market, and New Camelot; ride Moxim's rifts between continents. Start with the world guide, races, and OCCs.

About the game

Measured from the live mudlib (see docs/ASSESSMENT.md, 2026-07-13):

  • 62 playable races, 62 OCCs
  • ~116 spells, ~50 psionic powers, 158 skills
  • ~430 playable Rifts rooms across Praxis, Chi-Town, Tolkeen, Horton, Splynn, New Camelot, and more (world guide with honest zone status)
  • 173 mortal commands; custom prompt with MDC/SDC/PPE/ISP pool tokens (prompt, prompt tokens)

In-game help is two-level: help lists categories; help <category> lists topics; help <topic> opens a topic (separator-insensitive: armor of ithan matches armor_of_ithan). Use help index for the flat list.

Run your own copy

AetherMUD builds from source on Linux (Fedora, Ubuntu, WSL2). Short path: clone the repo, ./mud.sh init-config, build the FluffOS driver (./configure nm3 && make && make install), then ./mud.sh start and telnet to port 1122. Full steps: docs/INSTALL.md in the repository; daily operations in docs/RUNNING.md.
