Admin

Position: arch / head arch. Role tag: admin.

Tools

  • Staff of Demotion: promote, demote
  • Staff of Dominion: domain (includes apprentice submissions)
  • Staff of Creation: create, modify, add, delete, about, more, clone, purge, review
  • RP Skill Tool and Tattoo Gun

Promotion

makewiz <player>
setrole <player> admin|domain|coding|rp

makewiz creates an Apprentice with a kit. setrole assigns the permanent role after their track task is marked ready.

Reload

  • warmboot reloads daemons, cmds, obj, and /domains/adm; notifies online admins
  • ./mud.sh restart for /std/ or simul-efun changes

Run your own copy

AetherMUD builds from source on Linux (Fedora, Ubuntu, WSL2). Short path: clone the repo, ./mud.sh init-config, build the FluffOS driver (./configure nm3 && make && make install), then ./mud.sh start and telnet to port 1122. Full steps: docs/INSTALL.md in the repository; daily operations in docs/RUNNING.md.
