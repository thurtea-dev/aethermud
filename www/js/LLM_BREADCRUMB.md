# LLM breadcrumb - www/js/

1. Read `/CLAUDE.md` at the repo root first.
2. Read `README.md` in this directory (above).
3. Check the parent `www/LLM_BREADCRUMB.md` (and the repo root's) before
   assuming this directory's gotchas are self-contained.
4. Related: `www/` for the HTML pages, `www/css/` for the stylesheet.

## Gotchas specific to this directory

- This directory is currently empty. If you're adding the first script
  here, check whether the site's pages actually need client-side JS at
  all before introducing a new dependency - the site has deliberately
  stayed static so far.
