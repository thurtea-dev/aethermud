Character creation

Login is account-based: one account name and password, then a character select menu. Each account may have up to 5 characters. Creation itself happens in /domains/Praxis/setter.c with the chargen_guide NPC. Pool: 62 races, 66 OCCs, 158 skills (electives at chargen).

Login and character select

  1. Account name (create a new account on first connect, or use an existing one)
  2. Password
  3. Character select: pick a number for an existing character, or type new to create another (blocked at 5 characters for non-staff accounts)
  4. For a brand-new character: gender (male / female), then an optional display name

Chargen room steps

Every answer is typed as a plain word or name, with no command verb.

  1. Zone: americas, atlantis, or europe
  2. Attributes: roll, optional reroll (up to 4 rerolls; reroll stays legal until a race is chosen)
  3. Race: list, then the race name, e.g. human
  4. Alignment: the alignment name, e.g. scrupulous
  5. OCC: list, then the OCC name, e.g. headhunter, or none
  6. Elective skills: skills <category>, then the skill name, e.g. tracking
  7. Secondary skills (if granted)
  8. Finish: starting package, languages, teleport to the zone start room
