Quick Creation System

Staff of Creation verbs for rooms, NPCs, weapons, and armor. Work in your realm sandbox first.

Core verbs

create room|npc|weapon|armor <id>
modify <id> <field> <value>
add exit <dir> <dest>
delete <id>
about <id>
more <id>
clone <id>
purge <id>

Where files land

/realms/<you>/area/room/
/realms/<you>/area/npc/
/realms/<you>/area/weap/
/realms/<you>/area/armor/

Tips

  • Use short lowercase ids with underscores
  • Link exits both ways when rooms should be mutual
  • Spawn NPCs from room reset() with present() guards
  • Test with clone before promoting content into /domains/
