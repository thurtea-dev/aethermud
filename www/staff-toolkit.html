Staff Toolkit

Ambassador file tools and high-mortal commands: granted to every staff role (apprentice, rp_wiz, domain_wiz, creator/immortal, arch/head arch) on top of the role-specific tool covered on your own role page.

Ambassador commands

CommandWhat it does
earmuffsToggle blocking or accepting shouts.
gotoTeleport to a living thing by name, or to a file path if no living target is found.
homeReturn to your workroom, or another player's by name.
manLook up efun, SimulEfun, applies, and lfun documentation.
peopleFormatted, filterable, sortable roster of connected users.
set / unsetView, set, or clear a personal environment variable.

File tools

secure/cmds/ambassador: the shared read/write file kit every staff role gets.

CommandWhat it does
pwd / cdPrint your current working directory, or change it.
lsList a directory. Supports -a, -l, -t, -n, -b, -s.
cat / head / tail / moreDump a file, its first lines, its last lines, or page through it.
cpCopy a file to a new name or location.
diffLine-by-line diff of two files, paged through more.
edOpen the line editor on a file.
grepRegex search across one or more files. Supports > output redirection.
loadTry to load or compile an object from disk and report any errors.
sc / ssDeep-scan an object's full inventory tree, or shallow-scan its immediate inventory.
gaugeProfile a command's CPU and eval cost.

Note: gauge is reachable by every staff role in principle, but the command itself checks for the admin position specifically. In practice only admin can run it.

High-mortal commands

cmds/hm: granted to every staff role, and to any ordinary mortal above level 19.

CommandWhat it does
markMark the current room as a teleport anchor.
teleportTeleport to your marked or start location, or to another player of your own class.
startSet your current room as your login start point.
titleSet your post-name title string. No-ops for staff, who use setenv TITLE instead.
messageCustomize your room-entry and room-exit messages.
reviewDisplay your current message strings and title.
createConjure a weapon, armor, or bag from nothing.
cloakToggle a rogue stealth mode.
marry / divorceWed, or legally divorce, two players.

Conversion gap: most of this directory is still unconverted Nightmare III fantasy content. create and cloak check old-system skill percentages. marry and divorce only work for an admin or a player whose class is set to the legacy "monk" value, which no Rifts chargen ever sets, so in practice only admin can use them. Treat this whole directory as functional but not Rifts-converted.

Legacy skill commands

Most of cmds/skills/ is still Nightmare III legacy (about 70 files such as fireball, backstab, and heal): those verbs still gate on old fantasy classes (mage, cleric, monk, kataan, fighter, rogue) that no Rifts build has, so they stay live in the search path but inert for Rifts characters. Exceptions already converted to Rifts skills: limbs and replace (Medical Doctor), and lockpick (Pick Locks). Spell and psionic play for Rifts characters is still cast and psi, covered on the Commands page.
