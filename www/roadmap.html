Roadmap

Updated 2026-07-20. Source detail: docs/ASSESSMENT.md, docs/zone-expansion-plan.md, docs/playtest-checklist.md, CLAUDE.md.

AetherMUD, a TTRPG-inspired MUD. Core game, Fedora transfer package, apprentice kit, warmboot, Tolkeen, Horton wilderness, Chi-Town burbs/city, two-level help system, and staff www/ pages are in place. Live counts: 62 races, 66 OCCs, 115 spells, 51 psionics, 158 skills. What follows is remaining work in suggested order.

Now (ops and playtest)

  1. Copy production-aethermud/ to the Fedora Workstation mini PC. Follow internal/install-aethermud.md: verify SHA256, extract, rebuild the FluffOS driver, ./mud.sh start.
  2. Manual playtest pass from docs/playtest-checklist.md (password playtestaa for pt* chars):
    • Race eyeballs: dragons, vampires, atlanteans, titans
    • Cyber-Knight smoke on ptcknight
    • Walk Tolkeen (Moxim rift tolkeen) and Horton wilderness
  3. Promote wizard candidates while each is online: 
    makewiz ptdacon
makewiz ptcoder
makewiz ptrpwiz
    They use kit (domain / coding / rp). Domain reviews via domain option 10; coding via review. Then setrole <name> domain|coding|rp.
  4. Optional regression: python3 tools/section8_verify4.py with the MUD up.

Phase A (zone depth, highest ROI)

Deepen hubs players already reach. Exit criteria: a mortal can explore 2 to 3 hours from Praxis without looping the same twenty rooms.

  1. Done. Chi-Town burbs + fortified city (~51 rooms).
  2. Done. Tolkeen playable city (~29 rooms: core, siege edge, crypt).
  3. Done. Horton wilderness ring (~27 rooms total).
  4. Partial. Splynn Preserves hunting-grounds loop (~16 rooms) is open. Remaining: market services depth and deeper preserve combat sink.
  5. Optional polish: more Chi-Town / Splynn / NGR interiors if playtest finds thin spots.

Phase B (signature destinations)

  1. Stormshire / Federation of Magic v1 (~80 to 120 rooms). Hook via Moxim and Americas wilderness.
  2. New Camelot / NGR connective tissue (~+20 travel rooms, 1 to 2 side villages).
  3. Lone Star expansion (~+20 to 30: labs, kennels, escape routes).

Exit criteria: Chi-Town, New Camelot, Splynn, Stormshire, and Tolkeen each feel worth a mid-level visit.

Phase C (later / optional)

  • Africa (Phoenix Empire postcard)
  • Japan (tech / mystic contrast)
  • Australia
  • Deeper demon / dimensional planes beyond the current sliver

Start each as 15 to 25 rooms with one vendor, one threat, one unique loot. Do not block soft launch.

Systems and parity (parallel track)

  • Done. Chargen accept/reroll gate; no elective/secondary skill picking at creation; admin repairchar and makechar (see Admin commands and Chargen).
  • Done. Praxis Refuge Inn guest rooms (hotelroom101-110) and Survivors' Hostel (inn101-110); cemetery exists under domains/Praxis/cemetary/ (legacy spelling).
  • Content depth: vehicle interiors; more Palladium encyclopedia entries
  • Help: ~25 OCC help gaps (categories + separator-insensitive lookup already live)
  • CharGen: fuller gamemaster walkthrough NPC (status/help still available; deeper guided walkthrough remains)
  • Player-run shops
  • Website parity: in-game help browser over HTTP (who gateway already simplified)
  • Modern FluffOS cutover: paused under archive/; not a playtest blocker

Suggested sprint order

1. Fedora install + playtest checklist + wizard promotions
2. Splynn Preserves (+30)
3. Stormshire v1 core (+80)
4. Europe connectors + Lone Star (+40)
5. Systems/parity items as they block play
6. Phase C postcard zones when Phase B feels solid

Build rules (do not skip)

  • Spawns in reset() with present() guards; never NPCs in set_long
  • One purpose per area: combat, shopping, RP hub, or quest spine
  • Hook every new zone to Moxim and/or a walkable path from an existing hub
  • Preload-compile new rooms; clean log/catch before calling a phase done
  • Full reboot after /std/ or simul-efun changes; warmboot for daemons/cmds/domains
