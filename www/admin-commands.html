Admin Command Reference

Position: arch / head arch only. cmds/adm, secure/cmds/adm, cmds/creator, and secure/cmds/creator are only added to a player's command search path when their position is exactly arch or head arch. Coding wizards (position creator/immortal) do not get this directory, despite the name: see Coding wizard for what they build with instead.

cmds/adm

Player and character management

CommandWhat it does
advanceForce-advance a player N levels, rerolling HP/SDC/PPE/ISP gains.
giveClone an object by path and hand it to an online player.
grantrace / grantskillsBackfill race spells and psionics, or class skills, on a player. Post-hoc fix tools.
repaircharDiagnose or repair a broken character: dump chargen flags and stats; clear / finish stuck creation state; reroll racial attributes; skills to re-grant the class package; setstat for one attribute. Online player required. Does not replace setocc / setrcc.
fixdemoteOne-time repair for an offline player demoted before the offline restore path existed. Re-runs the full demotion restore on a named save.
severNarratively sever or restore a limb on an online player for RP (cybernetic stories, scars). Syntax: sever <player> <limb>, sever <player> restore <limb>, or sever <player> list. Does not deal combat damage.
makecharRebuild an online player without interactive chargen: makechar <player> <race> <occ|none> <alignment> with optional zone=, gear=default|none, skills=, spells=, psi=. Uses the same package grants as chargen finish.
setfactionSet or clear a player's faction membership.
setocc / setrccSet a player's class, or set their race and re-roll race stats.
setroleSet a wizard's position and role tag: admin, domain, coding, or rp.
setskillSet a named Rifts skill percentage on a player.
statDump a full stat block for an online player.
makewizPromote an online mortal to Apprentice wizard.
playerwipeWipe a player's save file, postal data, and realm directory.

World and movement

CommandWhat it does
gotoTeleport to a player's location or a room path.
transBring a player to your current location.
forceForce an online player to run an arbitrary command. Logged.
havenHide yourself from users(): not visible to who or finger.

Communication and broadcast

CommandWhat it does
archshoutShout without needing a language.
broadcastMessage every connected user, bypassing all blocking.
channelRestrict or unrestrict a player from a communication channel.

Logging, monitoring, and diagnostics

CommandWhat it does
monitorGet notified when a named player stops idling.
snoopersList who is snooping whom.
cache / fdinfo / opcprof / dumpallobjDriver cache stats, file descriptor status, opcode profiling, dump loaded objects.
commandsRegenerate the compiled command list from the creator, admin, and mortal command directories.
crefClone a referenced object (companion to the ref toolkit under secure/cmds/creator).

Game state

CommandWhat it does
endStart the Armageddon (scheduled shutdown) sequence.
warmbootBulk in-place reload of daemons, commands, and domains. Notifies online admins.
benchRun driver micro-benchmarks.

secure/cmds/adm

Account gatekeeping

CommandWhat it does
addguest / removeguest / whoguestsManage the guest list allowed in while the mud is locked.
allow / unallow / whoallowedLet a character name in from a registered site, or remove that allowance.
register / unregister / whoregisteredPut a site on, or off, the registration-required list.
tempbanPut a site on registration for a set number of minutes.
watch / unwatch / whowatchedFlag a name or site so its logins are logged.
arrestRemove a player pending law review.
ridPermanently remove a player. The safe way to delete a save file.

Grants, accounts, and logs

CommandWhat it does
grant / revokeGrant or revoke an RP title.
wizardApply the Wizard race and RCC package to a player (admin race).
link / unlinkLink character names together as one player's alts, or remove a link.
electionsStart or end elections, or set the next election date.
economyAdd a currency, or change an existing one's rate, inflation, or weight.
lawlogOpen or append a law-matters log for a player.
currencyBalance sheet of currency a player, or everyone, is holding.
setrebootSet the automatic reboot interval.
shutdownImmediate shutdown. Requires a logged reason.
unbanish / whobanishedRemove a name from the banished list, or list (or check) banished names.
xmotePromote or demote a player's level or position directly. Logged.

Refs and debugging toolkit

CommandWhat it does
callCall an arbitrary function on an object by reference.
dref / eref / ref / refs / unref / trefDestruct or otherwise operate on a referenced object. (cref lives under cmds/adm.)
dbxframe / dbxwhereInspect a stack frame, or print the last uncaught error trace for a player.
evalExecute arbitrary LPC inline. Extremely powerful; see callout below.

eval executes arbitrary LPC code inline. It is the single most powerful command in the game: whatever you can write, it can do, including destroying player data. Test on yourself or a disposable NPC before running it against a live player. Non-SECURE-group use is logged.
