# docs/reference/

Bulk Rifts reference material for developers - lookup tables and background
information, not step-by-step instructions.

- `information.md` - general Rifts/AetherMUD background reference.
- `invocations.md` - reference material on invocations/spell-like effects.
- `reference-chart.md` - the surviving OCC target list and related chart
  data (referenced from CLAUDE.md as the authoritative source now that
  the old helpfile/RCC-OCC-list text files are gone from the repo).
